# Voxie
A smart voice interaction system. We use 
- Flask - Back end
- Vue - Front end
- Gradio - GPT API Part

to develop this project.

# Run
There are 3 branch
- Main
- Front End
- Gradio APP

There is a build/deploy instrument in each of the branch. You need to start all of the app, the Voxie can be run.
